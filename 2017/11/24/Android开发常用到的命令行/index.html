Android开发常用到的命令行

Nov 24, 2017 | Hits

#ADB
adb start-server: 启动adb进程
adb kill-server: 杀死adb进程
adb tcpip 5555 : 重启adb服务，开启tcp指定端口 监听。使用adb connect 192.168.X.X连接Android设备会遇到connect refuse，使用这条命令会很有帮助。
adb connect <ip>： 连接对应ip的Android设备，连接成功之后可用于无线调试.可指定端口，默认是5555
adb pull <file1> <file2>: 导出设备中的file1文件 到 file2路径
adb push <file 1> <file2>: 导入file1到设备中file2路径
adb shell : 进入shell模式，可查看Android设备的文件，在设备上执行cmd等
adb shell dumpsys activity | grep "mFocusedActivity": 这个尝尝用来查看栈顶activity

#Linux

echo

echo -e "\x<十六进制数字>" <目标文件> : 向指定文件写入十六进制数据，添加-n参数可以指定不换行。写入数字为负值时，如：-128 (0xffffff80),取后两位即可。这样会导致一个问题，无法输入-1(oxffffffff)。
